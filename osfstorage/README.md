    <article class="markdown-body entry-content container-lg" itemprop="text"><h1><a id="user-content-open-access-the-effect-of-neurorehabilitation-on-multiple-sclerosis--unlocking-the-resting-state-fmri-data" class="anchor" aria-hidden="true" href="#open-access-the-effect-of-neurorehabilitation-on-multiple-sclerosis--unlocking-the-resting-state-fmri-data"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Open Access: The Effect of Neurorehabilitation on Multiple Sclerosis – Unlocking the Resting-State fMRI Data</h1>
<p>Crawled from <a href="https://osf.io/p2kj7/" rel="nofollow">OSF</a></p>
<h2><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Description</h2>
<p>We publish fMRI data of 60 people with Multiple Sclerosis before and after two months of neuroproprioceptive "facilitation and inhibition" treatment.</p>
<p><b>For more details on the original study design as well as the acquisition parameters, please consult our <a href="https://www.frontiersin.org/articles/10.3389/fnins.2021.662784/full" rel="nofollow">Data Report</a> </b> </p>

<hr>

<p>The data contain the following information:</p>

<p><code>clinical.csv</code></p>
<p>Clinical information about subjects.</p>
 <ul>
 <li> <b>ID</b>: subject identifier. </li>
 <li> <b>SEX</b>: "M" or "F" identifying males and females.</li>
 <li> <b>MS_TYPE</b>: type of multiple sclerosis either "remitting_relapsing", "primary_progressive" or "secondary progressive".</li>
 <li> <b>AGE</b>: age at the time of the study.</li>
 <li> <b>EDSS</b>: Expanded Disability Status Scale at the time of the study. </li>
 <li> <b>DATASET</b> whether the subject was a part of the pilot or the consecutive project (is relevant because of the slight change in the fMRI acquisition parameters).</li>
 <li> <b>DIAG_YEARS</b>: number of years since diagnosis.</li>
 <li> <b>BMI</b>: BMI at the time of the study.</li>
 <li> <b>THERAPY</b>: type of rehabilitation (see <a href="https://journals.lww.com/intjrehabilres/Abstract/2015/03000/Motor_programme_activating_therapy_influences.6.aspx" rel="nofollow">Prochazkova et al 2020</a>).</li>
 <li> <b>outliers_V1</b>: the absolute number of volumes flagged as outliers participant's first visit (<a href="https://web.conn-toolbox.org/home" rel="nofollow">step 3</a> of the CONN pipeline).</li>
 <li> <b>outliers_V2</b>: the absolute number of volumes flagged as outliers participant's second visit (<a href="https://web.conn-toolbox.org/home" rel="nofollow">step 3</a> of the CONN pipeline).</li>
 </ul>

<hr>

<p><code>nifti</code></p> 
<p>The directory contains <a href="https://web.conn-toolbox.org/fmri-methods/preprocessing-pipeline" rel="nofollow">CONN</a> preprocessed fMRI time-series. Each subject was recorded twice - before and after neuroproprioceptive "facilitation and inhibition" treatment. </p>

<hr>

<p><code>quality measures</code></p>
<p>Contains files with the appropriate displacement measures for each subject and visit.</p>
 <ul> 
 <li><code>DVARS.csv</code></li>
 <li><code>FD.csv</code></li>
</ul>

<hr>

<p><code>FC</code></p> 
<p>Contains functional connectivity matrices as <b>*.csv</b> for each subject and visit. Also contains file <code>AAL_labels.csv</code> with labels of AAL atlas regions.</p>

<hr>

<p>For more work of our <b>COBRA</b> group, please visit our <a href="http://cobra.cs.cas.cz/" rel="nofollow">website</a></p>.
